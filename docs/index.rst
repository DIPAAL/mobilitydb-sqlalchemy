`MobilityDB <https://github.com/ULB-CoDE-WIT/MobilityDB>`_ extensions for `SQLAlchemy <http://sqlalchemy.org/>`_.

Specifically, the following column types are provided, which can be used in SQLAlchemy models defined using the declarative API:

Installation
------------

.. code-block:: bash

    pip install mobilitydb_sqlalchemy

For getting started with MobilityDB SQLAlchemy, read our :doc:`quickstart`
